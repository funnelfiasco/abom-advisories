# ABOM Advisories

This is the source for advisories related to GitHub Actions used by [ABOM](https://github.com/JulietSecurity/abom).

## Get the list

The up-to-date advisory list is published as a [JSON file (./db/advisories.json)](./db/advisories.json), published at {{ site.time }}.
The ABOM of the action run to produce this version is available in CycloneDX format at [advisory-abom.cyclonedx.json](advisory-abom.cyclonedx.json).

## Contribute

Contributions — including corrections, updates, and new entries — are welcome at any time.
See the [Contributing section of the project's README](https://github.com/JulietSecurity/abom-advisories#contributing) for details.

The data presented in this advisory database is accurate to the best of our knowledge.
However, we do not make any guarantees as to the completeness or accuracy of the data.
Use at your own risk (and contribute fixes where something is wrong).
